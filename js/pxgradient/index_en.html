True Text Gradient (pixel by pixel)

Project at Github. Author: Sergey Nikitin · mrnix@ya.ru · twitter: @takoynik

Description and changelog

W

Has been updated (30.03.2012).

jQuery Plugin, that gradient colors in the text. Cross-browser (IE6++, Firefox3++, Opera9.64++, Chrome1++, Safari++). Saving the system font styles. You do not need images and special fonts. Regardless of background. The possibility of horizontal and vertical gradient. Pixel by pixel. Multiline support.

1.0.2 ver. (30.03.2012). New!

  • Now you do not need external css-file. All stylesheet is created by Plugin;
  • Significant refactoring. Faster and smaller.

Download pxgradient-1.0.2.zip with examples, 10Kb

Install

The plugin requires jQuery 1.3.2 or later

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script type="text/javascript" src="pxgradient-1.0.2.jquery.js"></script>

 

Usage:

<script type="text/javascript">
$(".text").pxgradient({ // any jQuery selector
  step: 10,
  colors: ["#fc0","#0fc","#00f"], // hex (#4fc05a or #333)
  dir: "x" // direction. x or y
});
</script>
